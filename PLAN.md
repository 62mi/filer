## Phase 1: 基盤＆低コスト高リターン ✅ 完了

> 触った瞬間に「このファイラー違う」と思わせる。日常使いできるファイラーの完成。

### 1.1 Undo/Redo スタック ✅
- [x] `undoStore` - Zustand による操作履歴スタック管理（最大50件）
  - 操作タイプ: `move` / `copy` / `rename` / `delete` / `create_dir` / `create_file` / `folderize`
  - 各操作の逆操作を自動実行（移動元/移動先、旧名/新名等）
- [x] Rust側: 既存の `commands/fs.rs` の各コマンドを Undo/Redo で再利用
- [x] Ctrl+Z（Undo）/ Ctrl+Y（Redo）キーバインド
- [x] `folderize` タイプ: フォルダ化のUndo時にファイル戻し + 空フォルダ削除まで実行

### 1.2 ファイル on ファイルでフォルダ化 ✅
- [x] D&D時にファイル同士の重なりを検出（`isFolderizeTarget` prop）
- [x] `group_files_into_folder` Rustコマンドで自動フォルダ作成 + 両ファイル格納
- [x] フォルダ名の自動提案（共通プレフィックス、ファイル名ステム）
- [x] シマーグラデーションアニメーション + フォルダアイコン変化でドロップ先を視覚表示
- [x] ドラッグ中の選択アイテム自身へのドロップ防止ガード

### 1.3 インラインリネーム進化 ✅
- [x] リネーム開始時に拡張子を除外して自動選択
- [x] Tabキーで次のファイルへ連続リネーム
- [x] Shift+Tabで前のファイルへ
- [x] 右クリックメニューからリネーム起動

### 1.4 ブックマークバー ✅
- [x] ナビゲーションバー下にブックマークバーを表示
- [x] アドレスバー右端の★ボタンで現在のフォルダをブックマーク追加/削除
- [x] ファイルリストからのD&Dでブックマーク追加（`effectAllowed: "all"`, `dropEffect: "copy"`）
- [x] クリックで一発ジャンプ、ダブルクリックでリネーム
- [x] ブックマーク同士のD&Dで並び替え
- [x] 右クリックメニュー: リネーム / フォルダ移動 / 削除
- [x] ブックマークフォルダ機能（作成/削除/リネーム、ドロップダウン表示）
- [x] localStorage 永続化（`BookmarkStore` with `Bookmark` + `BookmarkFolder` 型）

### 1.5 クイックルック (Space) ✅
- [x] Spaceキーでオーバーレイモーダルプレビュー（Mac風デザイン）
- [x] 画像: フルサイズ表示（`read_image_base64` Rustコマンド）
- [x] テキスト/コード: モノスペース表示（`read_text_file` Rustコマンド、最大100KB）
- [x] 矢印キー/←→で前後のファイルに移動
- [x] Esc で閉じる
- [x] Space 短押し: トグル開閉 / 長押し（200ms以上）: 離すと閉じる
- [x] 開閉アニメーション（scale 0.96→1、backdrop blur、GPU加速 `will-change` + `translateZ(0)`）
- [x] コンテンツ切替時のクロスフェードアニメーション

### 1.6 ドラッグゴースト ✅
- [x] `DragGhost.tsx` - カスタムドラッグイメージ
- [x] 複数ファイルのカードがファン状に重なる表示
- [x] 「+N」バッジで選択数を表示
- [x] アイコン（フォルダ/ファイル種別）付き

### 1.7 追加 UX 改善 ✅
- [x] Shift+Click / Shift+Arrow による範囲選択
- [x] 背景クリック / 他ファイルクリックで選択解除（Ctrl/Shift なし時）
- [x] マウスサイドボタンで履歴戻る/進む（button 3/4）
- [x] Alt+← / Alt+→ で履歴戻る/進む
- [x] Backspace で履歴戻る、Alt+Backspace で親フォルダへ
- [x] ファイル内検索機能（`search_files` WalkDir ベース、最大200件）
- [x] スムーズなディレクトリ読み込み（`smooth` パラメータでちらつき防止）

**Phase 1 完了:** Undo で安心して操作でき、フォルダ化・リネーム・クイックルック・ブックマークで基本操作が気持ちいい。「日常使いできるファイラー」完成。

## Phase 3: キラー機能（差別化の核）

> 他のファイラーにはない、唯一無二の機能。

### 3.1 ドラッグ時の移動先サジェスト ✅
- [x] ドラッグ開始300ms後にIntelliSense風ポップアップ
- [x] 移動履歴DB（SQLite）— 4軸スコアリング（ソースマッチ/拡張子/頻度/最近度）
- [x] ブックマーク・最近のタブ履歴も統合、MIN_SCORE閾値フィルタ
- [x] ドラッグ中のドロップでサジェスト先に移動
- [x] キーボードナビ（↑↓ Enter Esc）

### 3.2 フォルダルール（自動化） ✅
- [x] Rust ルールエンジン（条件: 拡張子/glob/名前含む/サイズ/経過日数、アクション: 移動/コピー/削除）
- [x] notify クレートによるフォルダ監視 + デバウンス + 自動実行
- [x] GUIルール管理ダイアログ（作成/編集/削除/トグル）
- [x] コンテキストメニュー「Folder Rules」からアクセス
- [x] ルール実行時のトースト通知 + ディレクトリ自動リロード

### 3.3 Rules.md + AI自動整理 ★★★★ → UXインパクト ★★★★★
- [ ] 自然言語でルールを記述（Markdown形式の `.rules.md` ファイル）
- [ ] AI（Claude API）が自然言語を解釈 → 既存のルールエンジン条件/アクションに変換
- [ ] ドライラン: 変換結果のプレビュー（対象ファイル一覧 + 実行予定アクション）
- [ ] 確認ダイアログ → ワンクリックで一括実行
- [ ] 実行結果のサマリー表示 + Undo対応

### 3.4 スマートフォルダ ★★★ → UXインパクト ★★★★☆
- [ ] 検索条件を保存して仮想フォルダ化

## 設計上の依存関係

- **Undo/Redo (#1.1)** は全てのファイル操作の安全網 → ✅ 実装済み
- **フォルダルール (#3.2) → Rules.md (#3.3)** はルールエンジンを共有
- **移動先サジェスト (#3.1)** は移動履歴DBが必要。Undo/Redoの履歴管理と統合可能
- **整理スコア (#4.4)** はフォルダルール + スマートフォルダの条件エンジンを再利用
