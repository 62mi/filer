## [1.2.0] - 2026-02-28

### Added
- ネイティブドラッグ&ドロップ — @crabnebula/tauri-plugin-drag によるOS標準D&D操作。ファイル移動・コピー・外部アプリ連携 (#45)
- ファイルサイズバー — リストビューのサイズ列に相対サイズの背景バーを表示。色は絶対サイズ帯（青<1MB/緑<100MB/橙<1GB/赤>1GB） (#26)
- 煩雑度スコア — ステータスバーにフォルダの整理度を★5段階+数値で表示。拡張子種類/ファイル古さ/数/ネスト構造の4項目で評価。Phase A即時計算+Phase B非同期Rust詳細計算の2段階 (#26)
- ホーム画面 — クイックアクセス・最近使用した項目・お気に入りタブ (#25)
- Windows Recent (.lnk) から最近使用したファイル取得コマンド `get_recent_files` (#25)
- Alt+Home ショートカットでホーム画面に移動 (#25)
- サイドバーにホームボタン追加 (#25)
- アドレスバーからターミナル起動 — cmd/powershell/pwsh/wt対応 (#25)
- ディレクトリ自動更新 — ファイル作成/削除/リネームをリアルタイム検知しUI自動反映
- ネイティブコンテキストメニュー — Tauri Menu APIベースのOSネイティブメニューに切替
- AI不要ファイル一掃プリセット — .tmp/Thumbs.db等の削除提案 + 削除確認UI
- カスタムタイトルバー・Explorer風タブUI (#24)
- Phase 2機能実装 + テストフィードバック修正 (#15, #17, #18)

### Fixed
- HICONリソース管理改善 — ビットマップ変換をエラー安全に (#12)
- AIファイル収集のメモリ最適化 + dead code削除 (#23, #11)
- expect()をエラーダイアログに変更し起動時panicを防止 (#22)
- Mutex lock unwrap()をunwrap_or_elseに変更しpanic防止 (#21)
- Windowsパス処理をヘルパー関数化し安全性改善 (#20)
- Biome lintエラー修正 — CRLF変換・useExhaustiveDependencies・biome設定調整 (#19)

### Changed
- コード品質改善 — マジック数字定数化・catch型安全性 (#11)
